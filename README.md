# Python Port Scanner

A lightweight, socket-based port discovery tool designed for network security auditing.

## Features
- **Fast Scanning:** Uses `socket.connect_ex` for efficient TCP port discovery.
- **Service Mapping:** Automatically identifies common services (SSH, FTP, HTTP, etc.).
- **User-Friendly:** Clean terminal output with timestamps.
